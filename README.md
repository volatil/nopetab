# NopeTab

Extension de Chrome para bloquear sitios distractores con reglas independientes por dominio.

## Cargar en Chrome

1. Ve a `chrome://extensions`.
2. Activa "Developer mode".
3. Haz clic en "Load unpacked".
4. Selecciona `C:\Users\paulo\Proyectos\nopetap`.

## Que incluye

- Reglas por sitio con soporte para varios dominios independientes.
- Bloqueos por fecha y hora exactas.
- Bloqueos recurrentes por horario semanal y dias de la semana.
- Popup que muestra el estado del sitio actual y su proxima regla.
- Pagina de opciones para configurar dominios, reglas y mensajes globales.
- Pantalla de bloqueo con desbloqueo de emergencia para la regla activa del sitio actual.
