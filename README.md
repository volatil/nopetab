# NopeTab

Extension de Chrome para bloquear sitios distractores durante intervalos de fecha y hora definidos manualmente.

## Cargar en Chrome

1. Ve a `chrome://extensions`.
2. Activa "Developer mode".
3. Haz clic en "Load unpacked".
4. Selecciona `C:\Users\paulo\Proyectos\nopetab`.

## Que incluye

- Popup para ver el estado actual y el proximo intervalo configurado.
- Pagina de opciones para configurar dominios, mensaje e intervalos de inicio/fin.
- Pantalla de bloqueo con desbloqueo de emergencia para el intervalo activo.
